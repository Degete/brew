# Brewfile

## Installation:

```sh
cd ~
git clone git@github.com:degete/brewfile.git ~/.brew
# Install all packages in `Brewfile`
brew bundle
brew bundle --file Brewfile.fonts
brew bundle --file Brewfile.apps
brew bundle --file Brewfile.cli
brew bundle --file Brewfile.dev
brew bundle --file Brewfile.devops
brew bundle --file Brewfile.game
```
